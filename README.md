# Breadcrumbman Atlas

A flagship companion app for the OSRS Breadcrumbman restrictive playthrough.

Breadcrumbman turns Old School RuneScape progression into a growing wiki-style graph. Start from a tiny seed, complete legal nodes, reveal linked content, rescue dead ends, and snowball from Lumbridge-scale beginnings to endgame routes.

## Core Loop

1. Choose a seed node.
2. Reveal connected breadcrumbs.
3. Complete one legal node.
4. Expand outward through linked quests, items, monsters, shops, transport, bosses, diaries, skills, and regions.
5. Use rescue mechanics when the graph gets stuck.

## Local development

```bash
npm install
npm run dev
```
